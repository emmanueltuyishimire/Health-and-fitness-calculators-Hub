# **App Name**: Body Balance

## Core Features:

- BMI Calculator: Calculate Body Mass Index (BMI) using height and weight.
- Body Fat Percentage Calculator: Estimate body fat percentage based on user input of measurements like waist, hip, and neck.
- Ideal Weight Calculator: Provide an ideal weight range based on user's height, gender, and body frame.
- Basal Metabolic Rate (BMR) Calculator: Calculate BMR to estimate the number of calories the body burns at rest.
- Daily Calorie Needs Calculator: Determine daily calorie needs based on BMR and activity level to help users manage their weight effectively.
- Personalized Recommendations: AI tool that recommends health and weight management content, providing personalized tips and guidance.

## Style Guidelines:

- Primary color: A calming light green (#A7D9A7) to represent health and growth.
- Background color: Very light off-white (#F5F5F5), close in hue to the primary, for a clean and spacious feel.
- Accent color: A light, warm orange (#E5B86B) to highlight key actions and information.
- Body and headline font: 'PT Sans' (sans-serif) for a modern and readable interface.
- Use clean, minimalist icons representing each calculator and feature.
- Implement a clean, responsive layout that is easy to navigate on both desktop and mobile devices.
- Subtle transitions and animations to enhance user experience, such as smooth scrolling and feedback on calculations.
